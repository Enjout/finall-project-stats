# Финальный проект по статистике — «ФармаПлюс»

Статистический анализ продаж аптечной сети: описательная статистика, 8 проверенных гипотез, визуализации и бизнес-рекомендации.

## Быстрый старт

```bash
pip install -r requirements.txt
python generate_data.py
python statistical_analysis.py
```

## Что сдавать

Основной документ: **[ОТЧЕТ_финальный_проект.md](ОТЧЕТ_финальный_проект.md)**

В отчёте есть:
- описание бизнеса и разреза анализа;
- подготовка данных;
- базовый анализ характеристик;
- 8 гипотез (разные типы данных и критерии);
- выводы и рекомендации;
- ссылки на код и графики.

## Структура

| Файл / папка | Назначение |
|--------------|------------|
| `generate_data.py` | Генерация синтетического датасета |
| `statistical_analysis.py` | Расчёты, тесты, графики |
| `data/` | Исходные CSV |
| `output/` | Результаты анализа (графики, таблицы) |
| `ОТЧЕТ_финальный_проект.md` | Итоговый отчёт для сдачи |

## Гипотезы

1. t-критерий — чек женщин vs мужчин  
2. Парный t-критерий — промо vs обычные дни  
3. ANOVA — чек по категориям  
4. χ² — оплата и возраст  
5. Корреляция Пирсона — цена и количество  
6. z-тест пропорций — онлайн-оплата Москва vs регионы  
7. Манна-Уитни — выручка в выходные  
8. Шапиро-Уилка — нормальность дневной выручки
